Short-Time Objective Intelligibility Measure

Matlab implementation of the Short-Time Objective Intelligibility (STOI) measure described in C.H. Taal, R.C. Hendriks, 
R. Heusdens, J. Jensen 'A Short-Time Objective Intelligibility Measure for Time-Frequency Weighted Noisy Speech', 
ICASSP 2010, Texas, Dallas

Downloaded by WSW July 18, 2014.
